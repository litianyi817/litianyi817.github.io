随笔 网站

欢迎来到我的个人网站！经过一番折腾，这个小小的空间终于上线了。

为什么要搭建个人网站

在这个社交媒体泛滥的时代，拥有一个完全属于自己的内容空间变得越来越重要。这里没有算法的干扰，没有流量的焦虑，只有纯粹的记录与分享。

我希望通过写作来整理自己的思路，把平时学习和实践中的收获沉淀下来。同时也期待能遇到志同道合的朋友，一起交流探讨。

这个网站会有什么

  • Blog：技术文章、项目复盘、学习笔记
  • Notes：碎片化的记录，灵感闪现时的快速笔记

技术选型

这个网站托管在 GitHub Pages 上，使用纯静态 HTML/CSS/JS 构建，没有任何框架依赖。选择这种方式的原因是：

  1. 极致的加载速度
  2. 零维护成本
  3. 完全掌控每一个细节
  4. GitHub Pages 提供免费的 HTTPS 和全球 CDN
